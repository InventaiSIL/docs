## What is InventAI?

InventAI is a toolkit for AI-powered game development. It allows you to:

- Generate 3D models and image assets in Unity.
- Create interactive Ren'Py games and manage stories online.
- Generate text and novel content.
- Manage characters and their interactions.

## How do I get started?

InventAI consists of multiple components. View the documentation for each one, depending on your use case :

- [InventAI Unity Library](docs/tools/unity) - Generate 3D models and image assets in Unity.
- [InventAI Novelcraft](docs/tools/novelcraft) - Create interactive Ren'Py games and manage stories online.
- [InventAI SDK](docs/sdk) - Core C# library for AI-powered game development (Godot, Unity, etc.)
